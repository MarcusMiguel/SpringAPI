
## ProductStore Application

![img](/src/main/resources/static/images/store2.gif)

## Project Structure
- **Frontend**: HTML page with Thymeleaf.
- **Backend**: Spring API secured with Spring Security + JWT, mostly controllers are for ADMIN usage only.

## Technologies
- Spring
- Spring Security
- JWT
- PostgreSQL
- Lombok
- Thymeleaf
